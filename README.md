# Employment Dashboard

A modern Employment Dashboard and ATS (Applicant Tracking System) prototype built using HTML, CSS, and JavaScript.

This project was designed as an internal HRMS + Recruitment Operations dashboard for:
- Recruiters
- Managers
- HR Teams
- Business Development Teams

---

# Features

## Overview Dashboard
- KPI cards
- Active cases summary
- Daily submissions
- Pending approvals
- Quick actions
- Activity tracking
- Upcoming events

---

## Active Cases
- Add job openings
- Track active/inactive cases
- Assign BD names
- Requirement management
- Export XLS functionality
- Search and filter support

---

## Daily Submissions
Recruiter workflow includes:
- Resume uploads
- Candidate pipeline tracking
- Submission tracking
- Interview stage management
- L1 / L2 / HR discussion stages
- Final offer tracking

Manager workflow includes:
- Submission review
- Approval system
- Employee working hours tracking
- Status updates

---

## Calendar Integration
- Manager calendar
- TA calendar
- Interview scheduling
- Meeting room booking
- Microsoft Teams integration concept
- Candidate interview invite management

---

## Profile Management
- Employee profile details
- PAN number
- Education details
- Mobile number
- Password change
- Profile photo section

---

## Leave Management
- Apply leave
- Remaining leave balance
- Overall leave tracking
- Leave policy section

---

## Employee Handbook
- Downloadable handbook
- Company policies
- Terms and conditions
- HR documentation
